安装mongodb

安装mongoldb

下载mongo4.0:

mongodb-linux-x86_64-rhel70-4.0.19.tg

解压mongodb:

tar -xvzf mongodb-linux-x86_64-rhel70-4.0.19.tgz

移动mongodb位置并修改名字：

mv mongodb-linux-x86_64-rhel70-4.0.19.tgz /home/ys/mongodb

创建data/db目录

mkdir -p data/db

创建mongo.conf文件：

touch mongo.conf

mongo.conf配置


mongod配置
#sharding:
#  clusterRole: shardsvr
#replication:
#  replSetName: shard1

  bindIp: 0.0.0.0

  path: /home/ys/logs/mongo.log
  logAppend: true

  fork: true

  dbPath: /home/ys/data/db/
  wiredTiger:
    engineConfig:
      cacheSizeGB: 1

#  keyFile: /var/mongodb/pki/keyfile

创建密钥文件并修改权限
openssl rand -base64 756 > <path-to-keyfile>
chmod 400 ~/appdb/conf/keyfile



mongos配置

  clusterRole: configsvr

  fork: true

  authorization: enabled
  keyFile: /var/mongodb/pki/keyfile

配置环境变量：

vi /etc/profile  

export MONGODB_HOME=/home/ys/mongodb  
export PATH=$PATH:$MONGODB_HOME/bin

安装完毕

启动mongoldb服务：

方法一：mongod -dbpath /home/ys/data/db  使用数据库启动
方法二：mongod -f mongo.conf     使用配置文件启动   （推荐）

连接monogdb客户端：

mongo

查看mongodb数据库：

show dbs

使用/创建数据库：

use test   --test表示已有的数据库或者创建的数据库

创建数据：

db.demo.insert({.....})   --demo表示创建的数据表

退出客户端：

exit

关闭mongodb服务方法:

方法1.在客户端关闭服务

use admin
db.shutdownServer()

方法2.在系统中关闭：

1.查看进程号：
ps -ef | grep mongo
2.杀掉进程
kill -6 进程号

如果要杀掉所有MongoDB进程：
pkill mongo

方法3.关闭服务

mongod -shutdown -dbpath /home/ys/data/db
